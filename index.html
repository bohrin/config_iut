The Dark Side

Ce mini projet a pour but de facilité l'utilisation de Linux sur les pc de l'iut.

Le principe est très simple, il suffit de lancer un tout petit script qui va totalement métamorphoser l'apparence et le comportement du bureau, tout en ajoutant de vrais outils pour travailler.

Installation

Il faut télécharger le script the_dark_side.bash (clique droit -> Enregistrer la cible du lien sous) et le mettre dans le net_home ou une clef usb. Après il ne reste plus qu'à l'exécuter et le tour est joué ! La magie vaudou prend (normalement) moins d'une demie minute pour faire son effet.

Les changements

On peut classer les changements apportés en deux catégories:

  • Modification de l'environnement de bureau
  • L'ajout de l'éditeur de texte Atom

Environnement de bureau

Ce qui saute d'abord aux yeux c'est le nouveau thème Arc tellement trop dark.

Ajout de raccourcies vers les applications les plus utilisées sur le bureau:

  • firefox/chromium
  • atom
  • scilab

Atom

Atom est un éditeur de texte qui a un système de paquet (plugins) qui lui permet d'être totalement personnalisable et donc parfaitement adapté à n'importe quelle situation.

BASH

linter-shellcheck: Affiche les erreurs à la volé grâce à shellcheck.

BDD

data-atom: Executer les requêtes sql sans quitter atom (au revoir phppgadmin)

C++

linter-gcc: Affiche les erreurs de compilation.

Web

atom-bootstrap3: Ajoute l'auto-complétion pour bootstrap 3.

atom-html-preview: Affiche la page web dans atom et elle s'actualise toute seule !

color-picker: Permet de sélectionner une couleur sans se casser la tête.

linter-htmlhint: Affiche les erreurs html.

linter-stylelint: Affiche les erreurs css.

random: Donne des noms, phrases, paragraphes et plein d'autre trucs au hasard.

webbox-color: Affiche les couleurs derrière les codes couleurs.

Autre

git-plus: Pour faire les commandes git depuis atom.

script: Lance script bash et compile/lance fichier cpp. Attention !: saisi utilisateur non prise en charge.
